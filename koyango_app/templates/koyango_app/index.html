Django app "Koyango"


This is a simple Django based app, which has 3 main functionalities - Calendar, Notes and World Time.

Use the navbar above to navigate between each functionality.


Calendar only shows the days for selected month.


World Time shows current time for popularly used time zones.


Notes allow for storing notes and assigning them a category.
